# Restaurant Website

A modern, responsive restaurant website designed to provide an engaging user experience across desktop, tablet, and mobile devices.

## Live Demo

🔗 **Website:** https://yuvarajthilak.github.io/Restaurant/

## Overview

This project showcases a contemporary restaurant landing page with a clean user interface, smooth navigation, and responsive design principles. It is built using modern front-end web technologies and optimized for performance and accessibility.

## Features

- Responsive design for all screen sizes
- Modern and intuitive user interface
- Interactive navigation
- Hero section with call-to-action
- Menu showcase section
- About section
- Contact section
- Optimized layout and styling

## Tech Stack

- HTML5
- CSS3
- JavaScript
